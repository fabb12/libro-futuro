📖 Il mio libro — impostazioni

Per leggere il libro non serve nulla: si apre da solo. Il token GitHub serve solo per salvare le modifiche sul repository.

Come creo il token?
  1. Vai su github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens → Generate new token
  2. In Repository access seleziona solo fabb12/libro-futuro
  3. In Permissions → Repository permissions imposta Contents: Read and write (per salvare le modifiche) e, se vuoi usare Genera PDF, anche Actions: Read and write
  4. Genera e incolla qui il token. Resta salvato solo su questo dispositivo.

Senza token puoi comunque leggere tutto il libro, ma non salvare le modifiche.

Note condivise (avanzato)

Di default le note che si aggiungono al testo restano solo su questo dispositivo. Per renderle condivise con tutti i lettori (chiunque scrive, tutti vedono) serve un progetto gratuito Firebase:

  1. Crea un progetto su console.firebase.google.com e attiva Firestore Database.
  2. Nelle Regole consenti lettura/scrittura sulla raccolta notes.
  3. In Impostazioni progetto copia ID progetto e Chiave API Web e incollali qui sotto.

Per attivarle per tutti in modo permanente, gli stessi due valori vanno messi nel file docs/notes-config.js del repository.
